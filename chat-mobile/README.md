# Мобильный клиент (React Native, Expo)

Экранная структура повторяет знакомый UX WhatsApp: вкладки «Чаты», «Контакты», «Профиль», экран отдельного диалога и упрощенная авторизация по номеру.

## Запуск
1. Установите зависимости:
   ```bash
   cd chat-mobile
   npm install
   ```
2. Укажите адрес API в переменной `EXPO_PUBLIC_API_URL` (например, `http://localhost:5000`).
3. Стартуйте Expo:
   ```bash
   npm start
   ```
   Затем выберите платформу (Android/iOS/Web) в интерфейсе Expo.

## Основные экраны
- **Login / VerifyCode** — ввод номера и кода `0000`.
- **Чаты** — список диалогов с последними сообщениями и временем.
- **Контакты** — пользователи системы, выбор создает новый диалог или открывает существующий.
- **Чат** — «пузыри» сообщений, мгновенные обновления через Socket.IO, статус «отправлено»/«доставлено» можно расширить.
- **Профиль** — имя, статус, телефон, обновление профиля и выход.

## Настройки и переменные
- `EXPO_PUBLIC_API_URL` — адрес backend (по умолчанию `http://localhost:5000`).
- Используются `AsyncStorage` для токена и `socket.io-client` для WebSocket.
