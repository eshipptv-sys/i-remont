# i-Remont

Монорепозиторий интернет-магазина и сервисного центра i-Remont. Клиент — React + Vite + Tailwind CSS, сервер — Express + Prisma + PostgreSQL.

## Новый модуль: WhatsApp‑like чат
- `chat-backend/` — легкий API + WebSocket сервер на Express/Socket.IO для чата с авторизацией по телефону и моделями User/Chat/Message.
- `chat-mobile/` — мобильный клиент на React Native (Expo) со знакомой структурой: чаты, контакты, диалоги и профиль.

Инструкции по запуску чата — в README соответствующих папок.

Полные инструкции по запуску и работе находятся в [docs/README.md](docs/README.md).
